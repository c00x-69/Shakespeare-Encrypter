{Shakespeare Encrypter}

QA:
-- What is it? ShakeSpeare Encrypter is an mini batch software, used to encrypt files / folders contents
-- How? By using cryptography python library.
-- Why? Can be used by someone's files. (WITH THEIR CONSENT!.. or not)
-- What is "Path:" , the question that the batch asks you and you need to specify a valid path from the pc it's runned


HOW TO RUN IT:
1. You need to install python (IF YOU DONT HAVE IT INSTALLED)

2. Need to install some depencies:
    pip install cryptography.fernet
    pip install cryptography
    pip install Fernet

3. To encrypt files in shakespeare you need to specify paths: 
    (eg.) "C:/Users/"  -- Will encrypt your /Users folder
    (eg.) "C:/Users/YOURUSER/Desktop/NewFolder" -- Will encrypt your /NewFolder from the desktop, you can specify any path you want
